Cal-C UI Calculator

This is a YouTube clone built using React, Tailwind CSS, and an API. The project aims to recreate the core functionality of YouTube, allowing users to search for videos, view video thumbnails and details, and watch videos.

  • Sleek and Responsive Design : Our Calculator combines modern aesthetics with responsive design, ensuring a seamless and visually pleasing experience on a wide range of devices, from smartphones to desktops.
  • Real-time Calculation : The core functionality of the Calculator allows users to perform arithmetic operations like addition, subtraction, multiplication, and division in real-time, with a large and easily readable display.
  • Dynamic Tilt.js Animations : Tilt.js animations bring a dynamic element to the calculator, allowing buttons and elements to respond to the user's cursor movements, adding a layer of interactivity and playfulness to the user experience.
  • Canvas Bloom Animations : Delight in the vibrant and mesmerizing Canvas Bloom animations that provide a visually stunning and immersive experience. The calculator interface is accentuated with smooth and responsive canvas animations, enhancing user engagement.
  • Cross-browser Compatibility : Our Calculator is compatible with major web browsers, ensuring a consistent experience for all users.
